# 思路
1. 读取csv文件数据：
在GetArray函数中，需要哪个csv文件的数据就解开相应段落的代码
2. 得到物体像素坐标
* 外参矩阵（4 * 4）* 物体世界坐标系齐次矩阵（4 * 1）= 相机坐标系齐次矩阵（4 * 1）
* 内参齐次矩阵（3* 4）* 外参矩阵的逆（4 * 4）* 物体世界坐标系齐次矩阵（4 * 1）=Zc * 物体像素齐次坐标
3. 对物体像素坐标进行筛选：0< x <1280 && 0< y <1024 满足条件则在大小定义为1280*1024的Mat图像上画出
